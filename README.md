# LocalBiz UGC AI 🚀

AI-powered UGC ad playground that generates Instagram Reels hooks, full ad scripts, and conversion-focused CTAs for local businesses.

## 🌐 Live Demo

https://localbiz-ugc-ai.lovable.app

## 📌 Project Overview

LocalBiz UGC AI is a modern AI SaaS-style web application built for Prompt Engineering Task 2 (2026) by Future Interns.

The platform helps local businesses instantly generate:

- 🎯 Scroll-stopping hooks
- 🎬 UGC ad scripts
- 📣 High-converting CTAs
- 📱 Platform-specific marketing content

using AI-powered prompt engineering.

The app is designed for:
- Cafés
- Gyms
- Salons
- Clinics
- Florists
- Creator brands
- SaaS startups
- Local businesses of any kind

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/localbiz-ugc-ai.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
